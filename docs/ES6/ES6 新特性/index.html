ES6 新特性

2. Map​

ES6 提供了 Map 数据结构。它类似于对象，也是键值对的集合。但是“键” 的范围不限于字符串，各种类型的值（包括对象）都可以当作键。 Map 也实现了 iterator 接口，所以可以使用『扩展运算符』和『for…of…』进行遍历。 Map 的属 性和方法：

  1. .size 返回 Map 的元素个数
  2. . set 增加一个新元素，返回当前 Map
  3. . get 返回键名对象的键值
  4. . has 检测 Map 中是否包含某个元素，返回 boolean 值
  5. . clear 清空集合，返回 undefined 
//创建一个空的Map
let m = new Map();
//添加元素
m.set("name","满")
m.set("change",function(){
    console.log("哈喽哈喽")
})
let key = { school:"尚硅谷"}
m.set(key,["北京","上海","深圳"])
console.log(m.size)  //3

//删除元素
m.delete("name")
console.log(m)
//获取元素
console.log(m.get("change"))
console.log(m.get(key))
//清空
m.clear()
//遍历
for(let v of m){
    console.log(v)
}

4. 数值扩展​

二进制和八进制

  1. ES6 提供了二进制和八进制数值的新的写法，分别用前缀 0b 和 0o 表示。
  1. Number.isFinite() 与 Number.isNaN() , Number.isFinite() 用来检查一个数值是否为有限的
  2. Number.isNaN() 用来检查一个值是否为 NaN
  3. Number.parseInt() 与 Number.parseFloat() ES6 将全局方法 parseInt 和 parseFloat，移植到 Number 对象上面，使用不变。
  1. Math.trunc 用于去除一个数的小数部分，返回整数部分。

5. 对象方法扩展​

ES6 新增了一些 Object 对象的方法

  1. Object.is 比较两个值是否严格相等，与『===』行为基本一致（+0 与 NaN）
  2. Object.assign 对象的合并，将源对象的所有可枚举属性，复制到目标对象
  3. proto、 setPrototypeOf、 setPrototypeOf 可以直接设置对象的原型
//1. Object.is 判断两个值是否完全相等
console.log(Object.is(120,120))  //true
console.log(Object.is(NaN,NaN))  //true
console.log(NaN === NaN)   //false
console.log(Object.is(+0,0))  //true
console.log(Object.is(+0,-0)) //false

//2. Object.assign 对象的合并
const a={
  name:"man",
  age :23
}
const b={
  name:"yu",
  gender:"nan"
}
console.log(Object.assign(a,b))  //{name: 'yu', age: 23 , gender:"nan"} 有则覆盖

//3. 设置原型对象
const school = {
  name :"尚硅谷"
}
const cities = {
  xiaoqu:["北京","天津","上海"]
}
Object.setPrototypeOf(school,cities)
console.log(Object.getPrototypeOf(school));
console.log(school)  //将cities 设置为 school 的原型

6. 模块化​

模块化是指将一个大的程序文件，拆分成许多小的文件，然后将小文件组合起来。

1.模块化的好处​

模块化的优势有以下几点：

  1. 防止命名冲突
  2. 代码复用
  3. 高维护性

2.模块化规范产品​

ES6 之前的模块化规范有：

  1. CommonJS => NodeJS、 Browserify
  1. AMD => requireJS
  2. CMD => seaJS

3.ES6 模块化语法​

模块功能主要由两个命令构成： export 和 import。

  1. export 命令用于规定模块的对外接口
  2. import 命令用于输入其他模块提供的功能

引入方式​

<script type="module">
  //1.通用的导入方式
  //引入模块1
  import * as m1 from "./m1.js"
  console.log(m1)
  m1.teach()
  console.log(m1.school)
  //引入模块2
  import * as m2 from "./m2.js"
  console.log(m2)
  //引入模块3
  import * as m3 from "./m3.js"
  console.log(m3)
  m3.default.hello()
  
  //2. 解构赋值形式
  import {school,teach} from "./m1.js"
  console.log(school,teach)
  //重名可以使用 as 修改名字
  import {school as guigu,hello} from "./m2.js"
  console.log(guigu ,hello)
  //导入 默认暴露的 形式
  import {default as m3} from "./m3.js"
  console.log(m3)

  //3. 简便形式（针对默认变量）
  import m4 from "./m3.js"
  console.log(m4)
</script>

暴露模块语法​

//分别暴露
export let school = "man"
export function teach(){
    console.log("hhhh")
}
//统一暴露
let school = "尚硅谷"
function hello(){
    console.log("ggggg")
}
export {school,hello}
//默认暴露
export default {
    school:"尚硅谷",
    hello:function(){
        console.log("xxxxx")
    }
}
